Public APIs

All APIs currently default to double precision (Float64). Support for single precision (Float32) is considered experimental, as it has not yet undergone comprehensive testing and validation.

TrixiCUDA.LobattoLegendreBasisGPU — Type
LobattoLegendreBasisGPU([RealT = Float64,] polydeg::Integer)

Create a nodal Lobatto-Legendre basis for polynomials of degree polydeg, partially storing and operating on GPU arrays (CuArray). This basis is designed for GPU-accelerated discontinuous Galerkin spectral element method (DGSEM).

Experimental implementation

This is an experimental feature and may change or be removed in future releases due to ongoing performance optimizations.
