Legacy (AWS)

This project were implemented and tested on AWS EC2 instances. So far, types of EC2 instances used in this project include: p3.2xlarge (single-GPU) and g4dn.12xlarge (multi-GPU). For more information about the EC2 instances, please refer to AWS EC2 Instance Types.

The following information is about the environment of the EC2 instances used in this project.
